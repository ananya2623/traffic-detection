# 🚗 Real-Time Traffic Detection using YOLOv8

A real-time traffic detection system built using **Python**, **OpenCV**, and **YOLOv8** that detects and identifies different types of vehicles through a webcam or video stream. The project is designed to perform fast and accurate object detection, making it useful for traffic monitoring and surveillance applications.

---

## 📌 Features

- 🚘 Real-time vehicle detection using YOLOv8
- 📷 Supports live webcam as input
- 🎯 Detects multiple vehicle types simultaneously
- 📦 Bounding boxes with class labels and confidence scores
- ⚡ Fast and lightweight implementation using YOLOv8 Nano
- 🖥️ Easy to run and customize

---

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/Real-Traffic-Detection.git
```

### 2. Navigate to the project folder

```bash
cd Real-Traffic-Detection
```

### 3. Install the required libraries

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python main.py
```

The webcam will open and start detecting vehicles in real time.
